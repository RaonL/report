# Domain Locker

Domain Locker는 Render Free Static Site에 맞춘 초경량 도메인 자산 관리 도구입니다.

## Features

- 도메인 만료일 D-Day 계산
- SSL 인증서 만료일 수동 기록
- 만료 임박 / 주의 / 정상 상태 표시
- 등록기관, 네임서버, 담당자, 운영 메모 관리
- 브라우저 LocalStorage 저장
- CSV 내보내기 / 불러오기
- 샘플 데이터 불러오기

## Render Free 배포

Render에서 Static Site로 배포합니다.

- Build Command: 비움
- Publish Directory: `./`
- Root Directory: 비움

서버, DB, API 없이 브라우저에서만 동작합니다.

# Engineer Weekly Report

IT 엔지니어의 반복적인 주간업무보고 작성을 돕는 초경량 정적 웹 도구입니다.

## Features

- 금주 수행 업무 문장 생성
- 차주 예정 업무 문장 생성
- 업무 유형별 문장 조합
- 간단 / 보고용 / 기술중심 톤 선택
- 결과 복사
- LocalStorage 기반 입력값 저장

## Render Static Site

Render에서 아래 값으로 배포하면 됩니다.

- Service Type: Static Site
- Build Command: 비움
- Publish Directory: `./`
- Root Directory: 비움

서버, DB, API 없이 브라우저에서만 동작합니다.
